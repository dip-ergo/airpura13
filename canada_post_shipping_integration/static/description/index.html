Odoo Canada Post Shipping Integration
Odoo E-Commerce Extension Providing In Depth Integration With Canada Post

While using the module please ensure these things:

  • Valid Credentials.
  • None-Zero Amount Of Weight.
  • Valid Address For User/Partner [Street, City, Zip, State Code, Country Code, Phone ].
  • Valid Canada Address For Warehouse [Street, City, Zip, State Code, Country Code, Phone ].

This Extension Allows

  • Real-time Shipping Rate Calculation [From BackEnd as well Website Also].
  • Contract Shipping Label Generation.
  • Non-Contract Shipping Label Generation.
  • Live Shipment Tracking Number.
  • Generate Return Label for both Contract and Non-Contract Shipping.
  • Compatible With All Version Of Odoo.
  • Full Admin Control
    • Admin can Add Extra Service Charges on actual shipping price.
    • Admin can Define/Update New Service as per their preferences.
    • Admin can Define/Update Packaging Type and Pickup Type for Canada Post shipping.
    • Admin can Define/Update Allowed Shipping Methods and Unit of Weight for Canada Post shipping.

Setup & Configure

Installation

This module required the extension of BackBone Shipping Module.

Shipping Configuration.

After the successful installation of the Canada Post Shipping Module module you will have to click the Delivery Methods menu under Inventory ->Delivery->Delivery Methods.

Select Customer Type:

  • Custom : For Non-Contract Shipping.
  • Commercial : For Contract Shipping.
    • By selecting customer type you can manage Contact and Non-Contact shipping.

Return Label Configuration:

  • Check Generate Return Label to enable return label generation.
  • Check Return Label Accessible from Customer Portal to enable accessing of return label from customer portal.

We have set up the most frequently used values
But please look all of them and change according to your needs .

Fill these Canada Post Credentials.

  • Developement Username.
  • Developement Password.
  • Production Username.
  • Production Password.

Environment - Select "Test Environment" while testing the module flow. Turn into "Production Environment", when you go live and ready to create real billable labels.

Work-flow

Canada Post Price Calculator [On BackEnd By Admin]

After configuring the Canada Post Shipping Module module you will be able to see the Canada Post Delivery in Delivery Method List[Sale --> Sale Order->Create].

Now :

  • Select A customer.
  • Add product in sale oder line.
  • Click Add shipping to add shipping method.
  • Select Canada Post Delivery as delivery method for this sale order.
  • Click on Get Rate it will list you the delivery charges.
  • Now Click on Add it assign delivery charges in sale order.
  • You can also update the delivery cost after adding it to order.

Wizard for rate calculation, select delivery carrier Get rate and Add to order.

Update Delivery cost after adding to order by click Update shipping

Canada Post Price Calculator [On FrontEnd By Site User]

This module also support the feature of Price Calculation From Website .

On the odoo website user can see the Canada Post delivery option in delivery method List in their cart before making final payment.

NOTE:
This delivery charge is combination of actual delivery charges and percentage/fixed amount set in Canada Post delivery configuration.

Canada Post Label Generator

NOTE:

Prior to label generation enable packaging option and assign access of packaging.

NOTE: Firstly you will have to create package of picking before clicking the validate.

Now create package by selecting those picking line which you want to put in single package then click on put in pack .

In side the wizard select the Canada Post Package Type you want to use for this package.

Here you can also modify the package Shipping Weight and also add the cover amount and description of package for future reference.

Finally After package generation click on validate/send to shipper.

Shipment Label will attach on the message Box.

Now download the label from the message box.

You can Track your Package from Track button.

Canada Post Return Label Generation

This module also provide the feature of return label generation.
Thus after receiving shipment if customer want to return its shipment then he/she can download the return label from Customer portal after filing return to attach label on the package and deliver back to Warehouse.

  • Generate return from validated shipment.

  • Select the quantities to be return among the delivered quantities.

  • Select Canada Post Delivery as carrier for this return shipment.

  • Now create package by clicking Put in pack and then click Print Return Label to generate return label.

    Note: You can generate return label only for one package. Do not create more than one package for generation of return label.

  • Return label can be downloaded from the message box.

  • After generation of return label customer can download this return label fromcustomer portal if Return Label Accessible from Customer Portal field is checked in delivery carrier configuration.
